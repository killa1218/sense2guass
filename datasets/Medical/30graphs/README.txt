Focused Entailment Graphs for Open IE Propositions
==================================================

This package contains 30 annotated entailment graphs.

- The graph nodes (propositions) are in tab-delimited files in the "nodes" directory.

- The graph edges are in tab-delimited files in the "edges" directory. Each edge is represented as LHS > RHS.

- A more detailed description of the annotation process is also included.
